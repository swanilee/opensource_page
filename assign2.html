Get Started

EXPRESS 실습 & 공부 정리 내용


총 14 단계로 구성되며, 사진아래에 세부 설명이 적혀 있습니다.

저의 경우, visual code를 사용하였습니다.
하지만 terminal이 아닌 내부에서 run을 할 경우 처음 하나의 node파일만 실행이되어
이후에는 계속해서 terminal을 통해 실행을 하였습니다.

1.

[ 새로운 express 서버 만들기 ]


서버를 만들어봅니다.

먼저 clone을 위한 repository를 만듭니다.


2.

[ json 파일 만들기 ]


3.

[ 미들웨어 직접 만들어 사용하기 ]


미들웨어는 중간에 요청을 처리하고 라우터는 요청 패스에 따라 분기하여 처리 하여 줍니다.
이것들 실행하기 앞 Express 외장 모듈을 설치 및 실행합니다.
% npm install express --save


4.

[ 미들웨어 ( send 메소드 사용 json 데이터 전송) ]



5.

[ redirect 메소드로 페이지 이동 ]


6.

[ 파라미터 전달 및 응답받기 ]


7.

[ 미들웨어 사용하기 ]






우선 body-parser 모듈과, serve-static 모듈을 설치하여줍니다.
그렇게하여 body-parser 미들웨어를 사용하여
그러고 난 다음, 서버에서 지정해준 id, password를 전송하여줍니다.
static 모듈을 이용하여 이미지를 열어주는 과정까지 실습합니다.



8.

[ 요청 라우팅하기 ]



요청 패스에 따라 라우팅을 하였고,
로그인 테스트를 진행하였습니다.


9.

[ 미들웨어로 오류 페이지 보여주기 ]

express-error-handler 모듈, errorhandler 모듈 사용






우선적으로 express-error-handler 모듈과 errorhandler 모듈을 설치하여 줍니다.










URL 파라미터를 사용하여 로그인 테스트를 진행하였고
에러메시지를 처리하였습니다.


10.

[ 쿠키 처리하기 ]

cookie-parser 미들웨어를 설치하여 사용합니다.








응답 객체의 cookie() 메소드를 호출합니다.


11.

[ 세션처리하기 ]

express-session 미들웨어를 설치하여 사용합니다.










로그아웃 시의 세션 처리 및 다른 페이지에서 세션 정보를 확인합니다.


12.

[ 파일업로드 기능 ]

multer과 cors 모듈을 설치하여 줍니다.












업로드된 파일 정보를 확인 가능합니다.


13.

[ 파일 내려받기 ]







mime모듈을 설치후, 파일을 내려받습니다.
